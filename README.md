# Traditional UHF RFID Reader

This version is a traditional UHF RFID reader migrated to **GNU Radio 3.10**, **UHD 4.1**, and **Ubuntu 22.04**.

The module design is located in `gr-rfid/lib`, and the overall control and configuration are implemented in the `reader.py` files under `apps`.

This implementation is based on the following reference:
[https://github.com/nkargas/Gen2-UHF-RFID-Reader](https://github.com/Yulong97/Gen2-UHF-RFID-Reader)
