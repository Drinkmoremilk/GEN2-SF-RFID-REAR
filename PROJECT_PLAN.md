# 基于软件定义无线电 (SDR) 的智能 RFID 阅读器研究计划

## 1. 项目背景与摘要
本项目基于 `gr-rfid` 开源项目（已升级至 GNU Radio 3.10），旨在利用软件定义无线电（SDR）的灵活性，突破传统商用 RFID 阅读器在底层协议访问和信号处理上的限制。当前代码库实现了基础的 EPC Gen2 协议（FM0 编码，固定 Q 值防碰撞），但在抗干扰能力、协议自适应性和感知应用方面存在巨大的研究空间。

本项目计划从**协议优化**、**AI 增强物理层**和**无线感知**三个维度展开深入研究，旨在构建一个高性能、可扩展的科研级 RFID 实验平台。

## 2. 研究目标 (Research Goals)

### 目标一：协议栈增强与自适应防碰撞算法研究
当前实现使用了固定的 Q 值（`FIXED_Q`）和单一的 FM0 编码。本目标旨在实现符合 ISO 18000-6C 标准的完整协议栈，重点在于：
1.  **自适应 Q 算法 (Adaptive Q-Algorithm)**: 实现动态时隙调整，根据实时碰撞率优化吞吐量。
2.  **Miller 编码支持**: 增加对 Miller-2/4/8 编码的支持，以提升在嘈杂环境下的通信稳定性。

### 目标二：基于深度学习的物理层解码 (AI-Enhanced PHY)
SDR 能够提供原始的 IQ 信号，这为利用深度学习技术解决低信噪比（SNR）下的解码问题提供了可能。
1.  **数据集构建**: 利用 SDR 采集不同信道条件下（多径、干扰）的标签信号。
2.  **神经解码器设计**: 设计 CNN 或 RNN 模型替代传统的 `tag_decoder_impl.cc` 中的差分检测算法，提高解码灵敏度。

### 目标三：基于 RFID 的环境感知 (RFID Sensing)
利用标签反向散射信号中的信道状态信息（CSI），探索 RFID 标签作为无源传感器的潜力。
1.  **信道估计提取**: 从 `tag_decoder_impl.cc` 中提取精确的信道估计值 (`h_est`)。
2.  **感知应用**: 研究基于信号特征（相位、幅度波动）的微动检测（如呼吸监测）或物体定位。

## 3. 详细行动计划 (Action Plan)

### 第一阶段：协议完善与基准测试 (1-2 个月)
*   **任务 1.1 (动态 Q 值)**:
    *   **修改文件**: `gr-rfid/lib/reader_impl.cc`, `gr-rfid/include/gnuradio/rfid/global_vars.h`
    *   **行动**: 实现 Gen2 标准中的 Q 值调整逻辑。根据解码结果（成功/碰撞/空闲），动态更新 `Q` 值，并生成 `QueryAdjust` 命令。
*   **任务 1.2 (Miller 解码)**:
    *   **修改文件**: `gr-rfid/lib/tag_decoder_impl.cc`
    *   **行动**: 实现副载波解调逻辑，支持 Miller 编码的多种速率。
*   **任务 1.3 (基准评估)**:
    *   **行动**: 使用 `apps/reader.py` 在不同距离和标签数量下进行测试，建立吞吐量和读取率的基准数据。

### 第二阶段：AI 解码器开发 (2-3 个月)
*   **任务 2.1 (数据采集)**:
    *   **行动**: 修改 `reader.py`，启用 `file_sink` 保存原始 IQ 信号（`source` 数据）及对应的标签真值（Label）。构建包含不同 SNR 和干扰类型的训练数据集。
*   **任务 2.2 (模型训练)**:
    *   **行动**: 使用 Python (PyTorch/TensorFlow) 离线训练神经网络解码器。输入为 IQ 序列，输出为比特流或符号概率。
*   **任务 2.3 (在线推理集成)**:
    *   **修改文件**: `gr-rfid/lib/tag_decoder_impl.cc`
    *   **行动**: 利用 `ONNX Runtime` 或 `TensorRT` 将训练好的模型集成到 C++ 模块中，实现实时 AI 解码。

### 第三阶段：感知应用探索 (2 个月)
*   **任务 3.1 (特征提取接口)**:
    *   **修改文件**: `gr-rfid/lib/tag_decoder_impl.cc`
    *   **行动**: 增加一个新的输出端口或消息接口，实时输出每个读取成功的标签的 `h_est` (复数信道估计) 和相位信息。
*   **任务 3.2 (应用验证)**:
    *   **行动**: 部署标签阵列，通过分析相位变化序列，尝试恢复环境中移动物体的轨迹或振动频率。

## 4. 预期成果 (Expected Outcomes)

1.  **高性能开源平台**: 一个支持动态 Q 算法和 Miller 编码的 GNU Radio 3.10 RFID OOT 模块，填补社区空白。
2.  **数据集**: 发布的“SDR-RFID 原始信号数据集”，包含多种环境下的标签反向散射信号，用于通信和感知研究。
3.  **学术论文**:
    *   一篇关于 *AI 增强型 RFID 解码器* 的会议/期刊论文，展示在低 SNR 下优于传统解码器的性能。
    *   一篇关于 *基于商用标签的低成本环境感知* 的研究报告。
4.  **演示系统**: 一套可实时演示的 RFID 阅读与感知原型系统。

## 5. 所需资源
*   **硬件**: USRP N210/B210 或同等 SDR 设备，UHF RFID 天线，商用 Gen2 标签。
*   **软件**: GNU Radio 3.10, PyTorch, Wireshark (用于协议分析)。
